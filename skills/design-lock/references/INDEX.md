## 命名規則

`<三位數 ID>-<appname>-<screen>.png`，對應 `.md` 同檔名。

範例：
```
mobile-app-examples/finance/001-cashapp-home.png
mobile-app-examples/finance/001-cashapp-home.md
```

ID 全域遞增，跨產業不重複（便於索引）。
